| 软件             | 版本

| SpringSecurity  | 6.0.0+
|==========================

=== 安装

[maven,xml]
----
<dependency>
    <groupId>com.github.yingzhuo</groupId>
    <artifactId>spring-turbo</artifactId>
    <version>${spring.turbo.latest}</version>
</dependency>
----
